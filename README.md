# Four Seasons Guide Service

Static website migration for [fourseasonsguides.com](https://fourseasonsguides.com/) — Lake Texoma striper fishing, Red River trips, and duck hunting.

## Local preview

```bash
python3 -m http.server 8080
```

Visit http://localhost:8080

## Stack

Plain HTML, CSS, and a small JavaScript file for mobile navigation. No build step.
